# HuelPicker

A small tool to pick my daily Huel flavours for me!

Every morning at 8am, I'll get an email which will pick 5 unique items from:

- Salted Caramel
- Strawberry
- Banana
- Apple Cinnamon
- Chocolate
- Gingerbread
- Chocolate Cherry
- Berry
- Mocha
- Pumpkin Spice
- Vanilla
- Rhubarb and Custard
- Toffee
